# `@mpflow/plugin-slim`

> Mpflow 的小程序瘦身插件

## 说明

该插件用于给 Mpflow 的构架提供小程序瘦身功能的集成，功能与 ![瘦身工具](https://developers.weixin.qq.com/miniprogram/dev/extended/utils/miniprogram-slim.html) 相同

## 安装

```bash
mpflow add slim
# 或
mpflow add @mpflow/plugin-slim
```

## 使用

### 代码相似度检查

```bash
npm run slim:cpd
# 或
yarn slim:cpd
```

### 图片压缩

直接通过 `mpflow build` 即可使用
